            Excel Metadata Viewer is a lightweight online tool that extracts and displays detailed information from Excel files (.xlsx and .xls). It presents useful metadata such as the file name, file size, number of sheets, sheet names, and row/column counts. Users can view the metadata directly on the page and export the results as a downloadable TXT or PDF file. Ideal for researchers, data analysts, or anyone needing quick insights into Excel file structure without opening the file.
