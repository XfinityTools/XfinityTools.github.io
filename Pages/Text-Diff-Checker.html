The "Text Diff Checker" is a free web-based tool designed to help users visually compare two blocks of text or uploaded .txt files. It displays the original content side by side and highlights any differences line by line, making it easy to identify changes. The tool also includes a “Differences Only” section and allows users to download the comparison as a PDF.


        
Instruction Manual:


        
1. Accessing the Tool:

        
Open your browser and go to the Text Diff Checker tool page on the XfinityTools website. Once loaded, you will see two main text boxes, file upload options, and buttons for comparing and downloading.


        
2. Input Options:

        
You can use either of the following methods to input text:

        
    
            
  • Manual Entry: Type or paste the first text into the left textarea and the second text into the right textarea.
    • 
            
  • File Upload: Use the file input fields to upload two separate .txt files. The contents will automatically populate the text areas.
    • 
        


        
3. Comparing Text:

        
After entering or uploading your text, click the "Compare" button. The tool will analyze both inputs line by line and generate a comparison in the lower section of the page.

        
    
            
  • Text 1: Displays the left-side input with differences highlighted.
    • 
            
  • Text 2: Displays the right-side input with differences highlighted.
    • 
            
  • Differences Only: Shows only the lines that differ between the two inputs.
    • 
        


        
4. Highlighting:

        
Lines that differ between the two texts are highlighted for easy identification:

        
    
            
  • Red (removed) indicates content from Text 1 that was changed or deleted.
    • 
            
  • Green (added) indicates content from Text 2 that was added or modified.
    • 
        


        
5. Downloading Results:

        
Click the "Download PDF" button to export the comparison output for offline viewing or documentation.


        
6. Additional Notes:

        
    
            
  • Make sure the files you upload are in plain text (.txt) format.
    • 
            
  • For best results, ensure both texts are properly formatted with consistent line breaks.
    • 
            
  • All line numbers are automatically displayed alongside each line for reference.
